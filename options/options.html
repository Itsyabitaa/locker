Browser lock settings

Configure PIN, scope, auto-lock, and where the lock screen appears.

General

Master locking switch and whether every site or only listed domains are affected.

Chrome does not allow extensions to block the “Disable” switch on chrome://extensions. PIN protection applies to this settings page only.

Locked sites

When global lock is off, only these domains trigger the lock screen.

Example: m.youtube.com matches youtube.com. youtube.com.evil.com does not.

Security

PIN is stored as SHA-256 only. Minimum length: 4 characters.

Auto-lock

Re-show the lock screen after you stop interacting with the page.
